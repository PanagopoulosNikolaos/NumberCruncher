## Directory Summary
This directory contains the orchestration layer and user interface for the Countdown solver suite. It includes a shell script for concurrent execution and a graphical Python TUI.

## Documentation Index
- [`tui.py`](../../docs/src_docs/tui_py_doc.md) — Interactive terminal interface for bench-marking all solvers.
- [`run_all.sh`](../../docs/src_docs/run_all_sh_doc.md) — Orchestration script for concurrent multi-language execution.
