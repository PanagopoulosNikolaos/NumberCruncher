## Directory Summary
This is the root source directory for the multi-language Countdown solver. It organizes the core logic of each implementation by language and includes the user interface components.

## Documentation Index
- [`python/`](python/README.md) — Python-based recursive solver and CLI.
- [`haskell/`](haskell/README.md) — Haskell-based functional solver and binary.
- [`prolog/`](prolog/README.md) — Prolog-based logic-driven solver and SWI-Prolog scripts.
- [`tui_wraper/`](tui_wraper/README.md) — Orchestration scripts and Rich-based TUI for result comparison.
