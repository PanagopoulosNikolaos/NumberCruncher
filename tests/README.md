## Directory Summary
This directory contains the automated test suite for the Countdown numbers game solvers and orchestration components. It implements unit tests for the Python solver and integration/consensus tests for the Python, Haskell, and Prolog solvers. The test suite verifies execution constraints, handles various difficulties, and tests complex edge cases.

## Documentation Index
- [`test_python_solver.py`](../docs/tests_docs/test_python_solver_doc.md) — Unit tests for the Python solver implementation.
- [`test_haskell_solver.py`](../docs/tests_docs/test_haskell_solver_doc.md) — Integration tests for the Haskell compiled binary solver.
- [`test_prolog_solver.py`](../docs/tests_docs/test_prolog_solver_doc.md) — Integration tests for the SWI-Prolog solver.
- [`test_orchestration.py`](../docs/tests_docs/test_orchestration_doc.md) — Integration tests for the run_all.sh shell orchestrator.
